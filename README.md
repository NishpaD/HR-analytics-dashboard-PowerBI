# HR Analytics Dashboard (Power BI)

## Description
This project analyzes employee attrition using a dataset of **1,470 employees** to identify key factors influencing workforce turnover. The dashboard provides interactive visualizations to explore attrition trends across departments, age groups, salary levels, and job roles.



## Objectives
- Analyze patterns and drivers of employee attrition  
- Identify high-risk employee segments  
- Support data-driven HR decision-making  



## Dataset
- **Records:** 1,470 employees  
- **Attrition Count:** 237 employees  
- **Attrition Rate:** 16.1%  

**Key Attributes:**
- Department (HR, R&D, Sales)  
- Age Groups  
- Salary Bands  
- Job Roles  
- Years at Company  
- Education Field  



## Features
- Interactive filters for department-level analysis  
- Attrition analysis by age group, salary band, and job role  
- Visualization of attrition trends across years in company  
- Gender-based attrition comparison  



## Key Insights
- Highest attrition observed in **age group 26–35**  
- Employees in **lower salary bands (≤5K)** show significantly higher attrition  
- Roles such as **Laboratory Technicians and Sales Executives** have the highest turnover  
- Early tenure employees (0–2 years) show higher attrition rates  



## Tools & Skills
- Power BI  
- Data Visualization  
- Exploratory Data Analysis (EDA)  
- Data Cleaning  
- Data Modeling  



## Outcome
Developed an interactive HR dashboard enabling **data-driven analysis of employee attrition**, helping identify high-risk segments and supporting workforce retention strategies.
